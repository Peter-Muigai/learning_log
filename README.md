# Learning Log

A Django-based web application that helps users **track and organize their learning journey**. Users can create topics, add entries/notes, and manage their learning progress with a simple, intuitive interface.

## Live Demo

You can access the deployed app here: [https://learning-log-1-qa0f.onrender.com/]

---

## Features

- Create and organize **topics**  
- Add **entries/notes** under each topic  
- Clean, user-friendly interface  
- **Admin dashboard** for managing topics and entries  
- Fully deployed on **Render free tier**  

---

## Tech Stack

- **Backend:** Django 5.2  
- **Database:** SQLite (pre-migrated for free-tier deployment)  
- **Web Server:** Gunicorn  
- **Deployment:** Render
